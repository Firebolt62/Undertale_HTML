Fanbase

The Undertale fanbase is one of the main reasons why the game has remained popular over many years. The community is one of the most creative and passionate fandoms. Here are some of my favourite creations from the Undertale fanbase:

 

Bad Time Simulator

Bad Time Simulator is an emulator that can be played on your web browser with a keyboard. It emulates one of the most iconic fights in Undertale, with Sans the Skeleton. The link to play it is here, and the source code is on github.

 

Flowey's Time Machine

Flowey's Time Machine is an unofficial save editor for Undertale. You can change multiple aspects of your save file and even generate new ones. The link to use it is here.

 

Undertale mods

  • Undertale Connect - This is a online multplayer mod that works through Steam.
  • Colored Sprites - This mod adds color to the original black and white sprites. It adds a nice touch.
  • Delta Patcher - This is not really a mod, but a tool to help install them.
